Cube Solver Help - Using Dist3

To get better performance by using the Dist3 search array it must first be generated to depth 10 and saved to a file which can be done . Also, arrays Dist 1 & 2 must be generated to depth 9 and saved to files as described on the previous page. Run the solver with use_dist3=1 added to the URL as shown by this and select the files (including Dist3_10F.dat) by pressing the [Load Dist Files] button.

The Dist files can also be made using programs that are part of the C Solver (written in the C programming language), see instructions in the README file (section "Dist Files for JS Solver") in solver.tar.gz.
